# 🍽️ Food supply backend system

## Features

- ✅ Product and supplier management
- ✅ Inventory tracking
- ✅ Order processing
- ✅ User registration & authentication (JWT-based)
- ✅ Product review & rating system
- ✅ RESTful API with JSON responses

## Technologies Used

- Java 21
- Spring Boot 3.x
- Spring Data JPA (Hibernate)
- PostgreSQL
- Spring Security + JWT
- Gradle
- Docker
